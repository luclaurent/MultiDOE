SOURCE CODE 


0001 Toolbox MultiDOE
0002 =======
0003 
0004 This Matlab toolbox regroups many existing tools for generating sample points using many specific DOE techniques:
0005 
0006 * Full factorial sampling
0007 * Latin Hypercube Sampling 
0008 * Optimal Latin Hypercube sampling
0009 * Improved Hypercube Sampling
0010 * Random Sampling
0011 
0012 Many techniques proposed in this toolbox are based on 
0013 
0014 * the [R software](https://cran.r-project.org/). More precisely, the library [lhs](https://cran.r-project.org/web/packages/lhs/index.html) is used. 
0015 * the [IHS](https://people.sc.fsu.edu/~jburkardt/m_src/ihs/ihs.html) matlab's softwares proposed by [John Burkardt](http://people.sc.fsu.edu/~jburkardt/).
0016 * the [LHS](http://www.mathworks.com/matlabcentral/fileexchange/4352-latin-hypercube-sampling/content/lhsu.m) proposed by Budiman Minasny.
0017 
0018 *NOTICE*: the use of the sampling techniques based on the R software are
0019 only usable on a Unix OS.
0020 
0021 Features
0022 ------
0023 The MultiDOE toolbox are able to 
0024 
0025 * generate sampling
0026 * sort the sample points using many techniques
0027 * add points to an existing sampling (based on the LHS and IHS procedures of the R software)
0028 * normalize/renormalize the sample points to be well defined in the specified design space
0029 * call automatically the R software (on Unix/Linux/OSX systems only)
0030 * compute scores (minimal distances, discrepancy...), see for instance the [Franco's PhD thesis (2008)](https://tel.archives-ouvertes.fr/tel-00803107/)
0031 
0032

0034 Requirements
0035 ------
0036 The toolbox requires:
0037 
0038 * the [Statistics and Machine Learning Toolbox](http://fr.mathworks.com/products/statistics/) of the Matlab's software (for the `lhsdesign` command called using `LHSD` keyword, optional)
0039 * the R software installed on an UNIX/Linux/OSX system and the `R` command available on the command line (for obtaining LHS or IHS called using `IHS_R` or `LHS_R` commands, optional) 
0040 * the [lhs](https://cran.r-project.org/web/packages/lhs/index.html) and [R.matlab](https://cran.r-project.org/web/packages/R.matlab/index.html) packages installed on R (see for instance [How to Install an R Package](http://math.usask.ca/~longhai/software/installrpkg.html))
0041 * the [graphviz software](www.graphviz.org) (the `dot` command must be available on the command line) for building the documentation (optional).
